# SoulMeet

SoulMeet is the world's first clinically measured social health companion for UHNI, HNI and family offices. This repository contains the frontend web application built with React, TanStack Start, and Tailwind CSS.

## Development

You need Node.js and npm.

```sh
npm install
npm run dev
```

## Deployment on Vercel

This project is configured to be easily deployable on Vercel. 
Simply import the repository into your Vercel dashboard and standard settings will be automatically applied. 

## Built with

- [TanStack Start](https://tanstack.com/start/latest)
- [TypeScript](https://www.typescriptlang.org/)
- [React](https://react.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
